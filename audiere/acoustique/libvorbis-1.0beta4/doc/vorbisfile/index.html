VorbisFile Documentation

VorbisFile is a stdio-based convenience library for decoding vorbis bitstreams. It is the simplest and quickest way to go from file to raw PCM audio samples--if your decoder doesn't need to read from anything other than a file, use this. This document explains how to use the vorbisfile API in detail.

Acoustique is a very simple and focused API with a single task:  to decode
various types of byte streams into RAW PCM data.

It currently can decode MP3 (via MAD), Ogg Vorbis (via vorbisfile), MOD,
XM, IT, S3M (via MikMod), and WAV.

Acoustique is under the GPL.  See license.txt.
